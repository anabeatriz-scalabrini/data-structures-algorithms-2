# Data Structures and Algorithms 2

This repository contains the source code and advanced projects developed during the **Data Structures and Algorithms 2** course of the Computational Mathematics undergraduate program at UNIFESP.

## Topics and Implementations
The files in this repository focus on advanced sorting algorithms, time complexity analysis, and balanced tree structures using C:
* **Advanced Linked Lists:** Insertion algorithms calculating the number of element comparisons (`judge1`).
* **Divide and Conquer:** Merge Sort implementation adapted to count structural inversions/drastic drops in arrays (`judge2`).
* **Advanced Sorting (Quick Sort):** Implementation using the "median of three" pivot selection, customized with priority rules and a swap counter for a triage system (`judge3`).
* **Heaps and Priority Queues:** In-place Max-Heap construction and Heap Sort algorithm applied to word frequency filtering (`judge4`).
* **Balanced Trees (Weight-Balanced Trees):** Implementation of a custom self-balancing binary search tree based on node weights, including operations for insertion, deletion, and interval searching (`judge5`).

## Technologies Used
* C Language
* GCC Compiler
* Advanced algorithm complexity evaluation
